# A Lean Mechanization of Reversible Occurrence Nets
 
**Authors:** Dávalos and Melgratti
 
This project contains a formal mechanization in [Lean 4](https://leanprover.github.io/) of the theory of **Reversible Occurrence Nets** — a model of concurrent computation that supports backward execution.
