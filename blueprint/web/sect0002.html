1 Nets

1.1 Net and Marked net definitions

Definition 1 Net and Marked Net

Net, MarkedNet A net \(N\) is a tuple \(N = (\alpha , \beta , {}^\bullet {\_ }, \_ ^\bullet )\), where \(\alpha \) is a nonempty set of places, \(\beta \) is the set of transitions, and \({}^\bullet {\_ }, \_ ^\bullet : \beta \rightarrow \mathbb {N}^{\alpha }\) assign source and target to each transition. We consider only nets in which every transition both consumes and produces, i.e., \({}^\bullet {{\sf t}} \neq \emptyset \) and \({\sf t}^\bullet \neq \emptyset \) for all \({\sf t}\in \beta \). A marking \(m\) of a net \(N\) is a multiset of places, i.e., \(m\in \mathbb {N}^{\alpha }\). A marked net is a pair \((N, m_0)\) where \(N\) is a net and \(m_0\) is a marking of \(N\).

Definition 2 Minimal

minimal The minimal elements of a net are those with an empty preset; as all transitions are consuming, these are necessarily places.

Definition 3 Enabled transition

is_enabled Let \(N\) be a net. A transition \({\sf t}\) is enabled at a marking \(m\), denoted \(m\llbracket {\sf t}\rrbracket _{\{ \! \! \{ N \} \! \! \} }\), if \({}^\bullet {{\sf t}}\le m\).

1.2 Firing and firing sequences

Definition 4 Firing and firing sequence

firingfiring_sequence The firing of a transition \(t\) enabled at \(m\) is denoted \(m \llbracket t \rangle _{\{ \! \! \{ N \} \! \! \} } m'\), where \(m' = m - {}^\bullet {t} + t^\bullet \). For a (possibly empty) sequence \(s = t_1; \dots ; t_n\), the firing sequence \(m \llbracket s \rrangle _{\{ \! \! \{ N \} \! \! \} } m'\) is defined by the chain:

\[ m \llbracket {\sf t_1} \rangle _{\{ \! \! \{ N \} \! \! \} } m_1 \, \wedge \, m_1 \llbracket {\sf t_2} \rangle _{\{ \! \! \{ N \} \! \! \} } m_2 \, \wedge \, \cdots \, \wedge \, m_{n-1} \llbracket {\sf t_n} \rangle _{\{ \! \! \{ N \} \! \! \} } m' \]
Definition 5 Reachable markings and safe nets

reachablesafe The set of reachable markings of a net \(N\) starting from \(m_0\), denoted by \({\it reach}(N,m_0)\), is the set of all markings obtainable through some firing sequence, i.e., \( \{ m' \mid \exists s, \; m_0 \ \llbracket s \rrangle _{\{ \! \! \{ N\} \! \! \} } \ m' \} .\) A marked net is 1-safe (or simply, safe) if every reachable marking is a set, i.e., \(m \in {\it reach}(N, m_0)\) implies \(m \in 2^{\alpha }\).
