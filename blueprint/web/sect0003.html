2 Occurrence nets

Definition 6 Causal Dependency

immediate, causal Let \(N\) be a net. The immediate dependency relation \({\lt}_{\{ \! \! \{ N \} \! \! \} }\subseteq {(\alpha \cup \beta )\times (\alpha \cup \beta )}\) is defined by

\begin{equation} \label{eq:immediate_dependency} {\lt}_{\{ \! \! \{ N \} \! \! \} } \; := \; \{ (x,y) \mid (y\in \beta \wedge x\in {}^\bullet {y}) \vee (x\in \beta \wedge y \in x^\bullet )\} \end{equation}
1

We denote the transitive closure of \({\lt}_{\{ \! \! \{ N \} \! \! \} }\) as the causal relation \(\prec _{\{ \! \! \{ N \} \! \! \} }\), and its reflexive-transitive closure as the causal preorder \(\preceq _{\{ \! \! \{ N \} \! \! \} }\). A net is acyclic if its causal relation is irreflexive, i.e., there is no \(x \in \alpha \cup \beta \) such that \(x \prec _{\{ \! \! \{ N \} \! \! \} } x\).

Definition 7 Conflict

immediate_conflict, Conflict.conflict Two distinct transitions \(t_1, t_2\) are in immediate conflict if they share a common pre-place, i.e., \({}^\bullet {t_1} \cap {}^\bullet {t_2} \neq \emptyset \), denoted by \(t_1 \# _{0\{ \! \! \{ N \} \! \! \} } t_2\). The conflict relation \(\# _{\{ \! \! \{ N \} \! \! \} }\) is then defined over \(x, y \in \alpha \cup \beta \) if there exist transitions \(t_1, t_2\) such that \(t_1 \preceq _{\{ \! \! \{ N \} \! \! \} } x\), \(t_2 \preceq _{\{ \! \! \{ N \} \! \! \} } y\), and \(t_1 \# _{0\{ \! \! \{ N \} \! \! \} } t_2\).

Definition 8 Backward conflict

A place \(s\) has a backward conflict if it is produced by two distinct transitions: \(\exists t_1, t_2, (t_1 \neq t_2) \wedge (s \in t_1^\bullet ) \wedge (s\in t_2^\bullet )\).

Definition 9 Occurrence net

is_occurrence, OccurrenceNet.is_marked_occurrence A marked net \((N,m_0)\) is an occurrence net if it satisfies the following statements:

  • \(N\) is acyclic;

  • there is no self-conflict, i.e., \(\neg ({\sf t}\# _{\{ \! \! \{ N \} \! \! \} }{\sf t})\) for all transition \({\sf t}\);

  • there is no backward conflict;

  • \(N\) is 1-safe, i.e., marking is a set;

  • the initial marking is the set of initial places, i.e., \(m_0\) is minimal.

Definition 10 Concurrent

OccurrenceNet.concurrent Two distinct elements \(x, y \in \alpha \cup \beta \) are concurrent, denoted \(x \mathrel {co}_{\{ \! \! \{ N \} \! \! \} }y\), if they they are neither causally dependent (\(x \not\preceq _{\{ \! \! \{ N\} \! \! \} }y\) and \(y \not\preceq _{\{ \! \! \{ N\} \! \! \} }x\)) nor in conflict (\(\neg (x \ \# _{\{ \! \! \{ N\} \! \! \} }y)\)).

A multiset \(X\) consists of concurrent places if it has no duplicates and every pair of distinct places is concurrent.

Lemma 11

firing_sequence_preserves_concurrent Let \(N\) an occurrence net. If \(m\) is a concurrent place and \(m\llbracket t\rangle _{\{ \! \! \{ N \} \! \! \} }m'\), then \(m'\) is a concurrent place.

Theorem 12 Occurrence net is safe

occ_net_safe lem:firing_sequence_preserves_concurrent If a marked net \((N,m_0)\) is occurrence, then is 1-safe.
